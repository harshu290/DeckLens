# DeckLens

## Features

- Investment Readiness Scoring
- Competitor Discovery
- Due Diligence Analysis
- Risk Assessment
- Investment Thesis Generation
- VC Mentor Feedback
- Final Investment Verdict

## Tech Stack

- Machine Learning
- PDF Processing
